## 前言

creator 2.x推出了新的Material系统，1.x自定义shader的方式已完全不适用。

本项目由原1.x的shader升级为2.0版本材质系统，为开源社区共享自己的微薄之力。

另外，我在腾讯课堂开了一期直播课，用捕鱼案例讲解creator游戏开发入门，希望新入坑的同学可以关注。

**腾讯课堂地址： https://ke.qq.com/course/378768?tuin=3ce6693**
